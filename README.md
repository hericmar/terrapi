# TerraPi
Jednoduchý manažer terárií, akvárií a jiných prostředí s měřitelnými
fyzikálními veličinami.

## Doporučení
I přes to, že projekt obsahuje CMakeLists.txt, je doporučeno verzovat
vygenerovaný Makefile.
